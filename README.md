# Fraud Detection System using Machine Learning

## Project Overview
The Fraud Detection System is a Machine Learning-based application developed to identify fraudulent financial transactions. The system analyzes transaction patterns and predicts whether a transaction is legitimate or fraudulent, helping to reduce financial risks and improve transaction security.

---

## Objectives
- Detect fraudulent financial transactions using Machine Learning techniques
- Perform Exploratory Data Analysis (EDA)
- Apply feature engineering methods for better model performance
- Build and evaluate Machine Learning models
- Develop a Streamlit-based web application for fraud prediction

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
- Git & GitHub

---

## Dataset Information
The dataset consists of financial transaction records containing features such as:
- Transaction Type
- Transaction Amount
- Sender Balance
- Receiver Balance
- Fraud Labels

These features are used to analyze transaction behavior and identify fraudulent activities.

---

## Features of the Project
